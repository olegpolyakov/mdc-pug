Chips

Chips are compact elements that allow users to enter information, select a choice, filter content, or trigger an action.

  • Docs
  • Guide

Examples

Action chips

+mdc-chip-set
    +mdc-chip('One')(action)
    +mdc-chip('Two')(action)
    +mdc-chip('Three')(action)
    +mdc-chip('Four')(action disabled)

Input chips

+mdc-chip-set
    +mdc-chip('One')(input)#i1
    +mdc-chip('Two')(input)#i2
    +mdc-chip('Three')(input)#i3
    +mdc-chip('Four')(input disabled)#i4

Filter chips

OneTwoThreeFour
+mdc-chip-set()(filter)
    +mdc-chip('One')(filter selected)#f1

    +mdc-chip('Three')(filter)#f3
    +mdc-chip('Four')(filter)#f4
