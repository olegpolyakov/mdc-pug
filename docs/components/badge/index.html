Badge

A badge generates a number indicator to the top-right of its child(ren).

    Examples

    With an avatar

    +mdc-badge(4)
  +mdc-avatar()(src='https://placeimg.com/128/128/people')

    With a button

    +mdc-badge(4)
  +mdc-button('Button')(outlined)

    With an icon

    notifications
    +mdc-badge(4)
  +mdc-icon('notifications')

    With an icon button

    +mdc-badge(4)(inset)
  +mdc-icon-button('notifications')

    With text

    Text
    +mdc-badge(4)
  +mdc-typography('Text', 'body1')(element='span')
